## Setup

```
pip install tensorflow==2.0.0
pip install tensorflow-hub==0.12.0
pip install matplotlib
pip insatll tqdm
pip install argparse
```

## Usage

```
python stylize.py [path_to_content.img] [path_to_style.img] --save
```

Use `python stylize.py -h` to get help.
